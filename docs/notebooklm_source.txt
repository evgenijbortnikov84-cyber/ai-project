==========================================
1. PROJECT OVERVIEW
==========================================

Purpose:
AI automation project using LLM APIs. Built in Python.

Architecture:
- src/        — core application logic (entry point: src/main.py)
- prompts/    — reusable prompt templates using {placeholder} syntax
- tests/      — pytest test suite
- docs/       — project documentation
- .github/workflows/ — CI/CD via GitHub Actions (Python 3.11, runs on push/PR to main)

Environment:
- Secrets and API keys go in a local .env file (gitignored)
- python-dotenv loads .env automatically at startup

==========================================
2. COMMANDS
==========================================

Install dependencies:
  pip install pytest python-dotenv

Run all tests:
  pytest tests/

Run a single test:
  pytest tests/test_main.py::test_main_runs

Run the app:
  python -m src.main

==========================================
3. PROMPT TEMPLATE CONVENTION
==========================================

Prompt templates are stored in prompts/ as Markdown files.
Variables use {placeholder} syntax and are substituted before sending to an LLM API.

Example template structure:

  ## System
  You are a helpful AI assistant.

  ## User
  {user_input}

  ## Notes
  - Keep responses concise
  - Use structured output when possible

==========================================
4. CI/CD PIPELINE
==========================================

File: .github/workflows/ci.yml
Triggers: push or pull_request to main branch
Steps:
  1. Checkout code
  2. Set up Python 3.11
  3. Install: pytest, python-dotenv
  4. Run: pytest tests/
